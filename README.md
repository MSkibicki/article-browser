## General info
App created to practice react-redux, react-router. It fetches data from API. Styling is made using SCSS.
	
## Technologies
Project is created with:
* React version: 16.13.0
* React-Router
* Redux
* SCSS
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../article-browser
$ npm install
$ npm start
```
